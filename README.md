To run the software 
1) Download the Repository on your local system
2) Run main-processing.pde, which compiles flower.pde and music.pde
3) After that, run hands.py
4) Pinch and create fun patterns on screen


Hardware Requirements:
Webcam


Software requirements:
Processing link: https://processing.org/download
